Create a Notion integration
Go to notion.so/my-integrations and click New integration. Give it any name, select your workspace, and click Save.
2
Copy your integration token
On the integration page, click Show next to the Internal Integration Secret. Copy that value — it starts with ntn_.
3
Connect your database
Open your trade journal in Notion. Click the ··· menu in the top right, go to Connections, and add the integration you just created.
4
Copy your database URL
Open your trade journal in Notion and copy the full URL from your browser. Paste it below — we'll extract the ID automatically.

Map Your Columns

Tell EdgeVantage which columns in your Notion database contain each piece of trade data. Only Date and R Value are required — everything else is optional.

Required Fields
Date
The date the trade was taken.
Required
→
R Value
Profit or loss in R multiples (e.g. 1.5 or −1). Must be a number column.
Required
→
Optional Fields
Pair / Instrument
The currency pair or instrument traded (e.g. EURUSD, Gold).
Optional
→
Direction
Long or Short. Should contain values like "Long" / "Short" or "Buy" / "Sell".
Optional
→
Session
The trading session (e.g. London, New York, Asian).
Optional
→
Result Override
If you have a Win / Loss / Breakeven column, map it here. If blank, EdgeVantage derives result from R Value.
Optional
→
Entry Time
The time the trade was entered (e.g. 08:30). Text or formula column.
Optional
→
Notes / Trade Name
Any text notes about the trade. Often the page title or Name column.
Optional
→
Stop Loss Size
Size of the stop loss in pips or points. Number column.
Optional
→
Plan Executed
Whether the trade followed the plan. Should contain "Yes" / "No" values.
Optional
→
Setup Type
Your setup, pattern, or play for this trade (e.g. Break & Retest, Liquidity Sweep). Unlocks the Expectancy by Setup panel.
Optional
→
Each column shows its Notion property type to help you choose the right one. You can update this mapping at any time from the Settings menu in the dashboard.
